# php_3g_caller
> Автоответчик из 3g модема.

Пример взаимодействия с 3g модемом Huawei E171 на PHP



Проверка баланса:
```php
php balance.php
```

Исходящий вызов: 
```php
./call.sh 79*******15
```
79*******15 - номер абонента

```php
screen -q
```

Запуск демона ожидания входящего звонка:
```php
./start.sh
```
